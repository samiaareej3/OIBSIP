# OIBSIP Data Science Internship Projects

This repository contains Data Science and Machine Learning projects completed as part of the Oasis Infobyte Internship Program (OIBSIP).

## Projects

### 1. Iris Flower Classification
A machine learning model that classifies iris flowers into different species based on sepal and petal measurements.

**Technologies Used**
- Python
- Pandas
- Matplotlib
- Scikit-learn

---

### 2. Car Price Prediction
A regression-based machine learning project that predicts the selling price of cars using various vehicle attributes.

**Technologies Used**
- Python
- Pandas
- Matplotlib
- Scikit-learn

---

### 3. Sales Prediction Using Python
A machine learning model that predicts product sales based on advertising expenditure across different media platforms.

**Technologies Used**
- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## Tools & Libraries
