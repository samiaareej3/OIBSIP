# Car Price Prediction

## Objective
Build a Machine Learning model to predict the selling price of cars based on features such as year, fuel type, transmission, and kilometers driven.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Car Price Prediction Dataset containing information about various cars and their selling prices.

## Model Used
Random Forest Regressor

## Result
The model successfully predicted car prices with a high R² score, demonstrating strong prediction accuracy.
